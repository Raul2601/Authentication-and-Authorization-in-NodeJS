## Authentication and Authorization in NodeJS

## What's included
 - Frontend: simple Angular10 application
 - Backend: NodeJs with Express

Authentication:
 - Passport

Authorization:
 - Middleware
 - Roles and Permissions

## Start App 
Run backend
```bash
# go to backend folder
cd backend
# install all dependencies
npm install
# run app
node inde.js
```

Run fronend

```bash
#go to frontend folder
cd frontend
# install all dependencies
npm install
# build app
ng build
# run app
ng serve
```
CTRL+C to stop.
